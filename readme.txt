Browse and import demo sites with Customify with few clicks.

== Description ==
This plugin is an add-on for the Customify WordPress Theme.

Browse and import demo sites from your WordPress backend with 4 steps:
1. Choice a site to import.
2. Install recommended plugins.
3. Import media and content.
4. Import config.

Done!

== Installation ==
1. Upload the entire `customify-sites` folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the `Plugins` menu in WordPress.

You will find `Appearance -> Customify Sites` menu in your WordPress admin panel.
